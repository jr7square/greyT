iOS Application: Running, Brah?

Developed in XCode 8.0 with Swift 3.0 for the use of iPhone devices. 

App tracks distance traveled, can differentiate between stationary, walking and running. When user remains stationary for a while Shia Labeouf's "Just Do It!" motivational audio from comes up and will not stop unless the user starts running again. 

Import GitHub directory as an XCode project and run application through simulator.
